Task 2
Goal
Learn to distinguish between Write and WriteLine operators.

What needs to be done
The Write and WriteLine methods are very similar, but they have an important difference. The Write method does not wrap the line after the output to the screen has occurred. Develop an application that will show you the difference between these two operators:

Display the inscription Hello World!!! on the screen using the Console method.Write. 
Use Console.Write for each word (you will have three method calls: first for the word Hello, then for the word World, and then for three exclamation marks).
To prevent words from "sticking together", use spaces inside quotation marks.
Add a Console delay.ReadLine, so that the application does not close when the execution is completed.
Tips and recommendations
Please note: to close the application, you need to press the Enter key.

What is being evaluated
The program displays the inscription specified in the task and does not close until the Enter key is pressed.
