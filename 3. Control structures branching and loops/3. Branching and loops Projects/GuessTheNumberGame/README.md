Guess the Number Game

What to do:

Develop a program using the following algorithm:

The user enters the maximum integer number of the range.
Based on the range of integers (from 0 to "user input"), the program generates a random integer number from the range.
The user is prompted to enter the randomly generated number. The user inputs their guesses in the console of the application.
If the number is smaller than the guessed number, the program informs the user.
If the number is larger, the program also informs the user that the number is greater than the guessed number.
The program ends when the user guesses the number.
If the user is tired of playing, instead of entering a number, they input an empty string and press Enter. Then, the program ends, showing the number that was guessed.

Tips and recommendations:

To organize an endless input of numbers and give the user the opportunity to input as many numbers as they want, use a while or do-while loop with the condition (1 == 1) or simply (true).
Use break to exit the infinite loop.
